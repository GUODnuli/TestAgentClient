## 任务
评估检索结果:
1. 信息是否足以回答查询?
2. 置信度 (0.0-1.0)?
3. 如果不足，缺少什么?

## 评估标准
- 相关性: 检索结果与查询的相关程度
- 完整性: 是否涵盖查询的所有方面
- 具体性: 是否包含具体可用的信息
- 一致性: 检索结果之间是否一致

## 置信度参考
- 0.9-1.0: 非常充分，可以直接使用
- 0.7-0.9: 基本充分，可能需要少量补充
- 0.5-0.7: 部分充分，建议继续搜索
- 0.3-0.5: 不太充分，需要调整策略
- 0.0-0.3: 不充分，需要大幅调整

## 输出格式
输出 JSON:
```json
{
  "is_sufficient": true,
  "confidence": 0.85,
  "feedback": "需要更多关于 X 的信息",
  "reasoning": "评估理由..."
}
```
